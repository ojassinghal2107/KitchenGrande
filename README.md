# Kitchen Grande Website

Kitchen Grande is a full-stack web application designed to showcase elegant kitchen designs and services. Built with a modern tech stack, it combines a responsive frontend with a secure, modular backend architecture.

## 🔧 Tech Stack

**Frontend:**
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

**Backend:**
- Java
- Spring Boot
- Hibernate
- Spring Data JPA

**Database:**
- MySQL

## 🚀 Features

- Responsive UI with Bootstrap
- Dynamic kitchen design showcase
- RESTful APIs for data handling
- Secure authentication (JWT-ready)
- Modular backend architecture
- Evaluator-friendly structure and documentation
